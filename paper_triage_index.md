
1. Core Burnout Theory

->Burnout: A Review of Theory and Measurement
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC8834764/
- Purpose: Helps define burnout properly and explains how burnout is measured in research.

->The Copenhagen Burnout Inventory: A new tool for the assessment of burnout
- Link: https://doi.org/10.1080/02678370500297720
- Purpose: Useful for understanding a validated burnout questionnaire that separates personal, work-related, and client-related burnout.

2. Measurement Scales

->Mental fatigue measurement using eye metrics: A systematic literature review
- Link: https://pubmed.ncbi.nlm.nih.gov/33825234/
- Purpose: Explains how eye-related features can be used to measure mental fatigue.

3. Eye Blink, Eye Closure, and Fatigue Detection

->A Real-Time Fatigue Sensing and Enhanced Feedback System
- Link: https://www.mdpi.com/2078-2489/13/5/230
- Purpose: Useful for designing a real-time fatigue detection system using eye features and feedback.

->Evaluation of a Fatigue Detector Using Eye Closure-Associated Indicators Acquired from Truck Drivers in a Simulator Study
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC8512350/
- Purpose: Shows how eye closure indicators can detect fatigue.

->Adjusting eye aspect ratio for strong eye blink detection based on facial landmarks
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC9044337/
- Purpose: Useful for blink detection using Eye Aspect Ratio and facial landmarks.

->Driver fatigue detection based on eye state
- Link: https://pubmed.ncbi.nlm.nih.gov/26410512/
- Purpose: Supports the idea that eye state can be used for fatigue detection.

->Experimental study on visual detection for fatigue of fixed-position staff
- Link: https://pubmed.ncbi.nlm.nih.gov/28802427/
- Purpose: Useful for non-contact fatigue detection in people working in fixed positions.

->Real-time Eye Blink Detection using Computer Vision and Machine Learning
- Link: https://www.preprints.org/manuscript/202410.0131
- Purpose: Simple reference for real-time blink detection, but it should be used carefully because it is a preprint.

4. Student or E-Learning Engagement

->Video-based real-time monitoring of engagement in E-learning using MediaPipe through multi-feature analysis
- Link: https://www.sciencedirect.com/science/article/abs/pii/S0957417425018585
- Purpose: Useful if the project focuses on students, online learning, engagement, attention, or fatigue.

5. Tools and Feature Extraction

->OpenFace 2.0: Facial Behavior Analysis Toolkit
- Link: https://ieeexplore.ieee.org/document/8373812
- Purpose: Useful for extracting facial landmarks, gaze, head pose, and facial behavior features.
