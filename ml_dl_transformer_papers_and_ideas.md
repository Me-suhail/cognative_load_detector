1.Machine Learning for Burnout Prediction

->Using Machine Learning in Burnout Prediction: A Survey
- Link: https://link.springer.com/article/10.1007/s10560-020-00733-w
- Purpose: Supports the idea that machine learning can be used for burnout prediction using questionnaires, sensors, EEG, and other data sources.

->A machine learning model to predict the risk factors causing feelings of burnout and emotional exhaustion
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC11687012/
- Purpose: Shows that supervised machine learning can predict self-reported burnout and emotional exhaustion from survey data.

->Detecting Burnout Among Undergraduate Computing Students with University Records
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC12691878/
- Purpose: Useful if the project focuses on students; shows burnout detection using machine learning and academic or university data.
